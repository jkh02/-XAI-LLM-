## -XAI-LLM-
설명 가능한(XAI) LLM 기반 상담 챗봇:   입력 요인별 기여도 분석 및 시각화 시스템
# 1. 배경 및 필요성

 LLM 기반 법률·의학 상담 챗봇이 확산되고 있지만, 사용자는 챗봇이 제시한 결론(예: 고소 가능성, 병원 방문 필요성 등)에 대해 왜 이런 판단이 나왔는지 이해하기 어렵고, 신뢰성 확보가 어려움

 설명 가능한 인공지능(XAI, Explainable AI)은 머신러닝 모델이 생성한 결과와 판단 과정을 인간이 이해하고 신뢰할 수 있도록 만드는 기술을 의미함. 즉, 모델의 예측 결과에 대한 이유, 근거, 기여도를 사람이 해석 가능한 형태로 제공하는 것이 핵심임

 특히 상담 상황에서는 여러 입력 정보(예: 피해 금액, 증거 유무, 경과 시간, 과실 정도 등)가 결론에 복합적으로 작용하므로, 각 요소가 최종 판단에 얼마나 영향을 미쳤는지를 정량적으로 제시할 수 있는 XAI 기능이 매우 중요함. 본 프로젝트는 LLM 챗봇의 판단 결과에 대해 ① 입력 변수별 분석, ② 기여도 시각화, ③ 자연어 해설을 제공하는 XAI 기반 설명 모듈을 구현하는 것을 목표로 함



#2. 세부 연구 내용

 LM 상담 챗봇 구현 

 법률, 의학 등 도메인을 선택하고, 상황 변수와 사용자의 서술 정보를 기반으로 위험·가능성 판단과 요약 답변을 생성하는 LLM 챗봇을 구축함

 입력 요인별 책임도(XAI) 산출

 입력 변수 하나씩 변경하여 LLM 판단 변화량을 계산하여 기여도 분석을 수행함

 다수 사례에 대한 LLM 출력 데이터를 모델에 학습시켜 SHAP 기반 중요도를 산출함

 두 방식의 결과를 비교하여 안정성·계산 비용·해석 용이성을 평가함

 책임도 시각화 및 설명 제공 

 변수별 기여도(%)를 시각화(막대·도넛 차트 등)하여 어떤 요인이 결론에 가장 영향을 주었는지 명확히 제시함

 기여도가 큰 이유를 간단한 자연어 설명 형태로 제공함
